#### Run the Turtlesim Node
Open a console and run the Turtlesim Node with the following command
```
ros2 run turtlesim turtlesim_node
``` 

#### Run the Agent
Open a console, go to the "iot-agent/lib" folder and run the following command
```
node index.js
```
